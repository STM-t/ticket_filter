# 🎫 Интеллектуальная система фильтрации заявок техподдержки

## ✨ Особенности

- 🤖 **Реальная модель машинного обучения** (Logistic Regression + TF-IDF)
- 🔄 **Двухэтапная классификация**: группа отдела → конкретный отдел
- 🧠 **Предобработка текста**: лемматизация (pymorphy2), удаление стоп-слов, TF-IDF векторизация
- ⚡ **Расчёт приоритета** на основе иерархии пользователя
- 🌐 **Веб-интерфейс** на Flask с адаптивным дизайном
- 📊 **Метрики качества**: accuracy, precision, recall, F1-score
